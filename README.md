# Teste de Frontend

### Executar

* Baixar o repositório
* NodeJs instalado
* Executar o comando
```
npm install
```

para instalar as dependencias

configurar o arquivo .env com a URL das APIs Backend encontradas em: https://github.com/vipdesenvolvimento/teste-backend

depois executar o comando:

```
npm start
```
